# Medicore

A real-time healthcare communication platform connecting patients with healthcare workers in their districts.

## Key Features

- **Real-time Chat**: Instant secure messaging between patients and healthcare professionals.
- **District-Based Routing**: Automatically connects patients to available workers in their specific health district.
- **Task Management**: Healthcare workers can create and manage tasks to track patient needs.
- **File Sharing**: Securely share medical documents and images.
- **Role-Based Access**: Dedicated interfaces for Patients, Healthcare Workers, and Administrators.
- **Secure Authentication**: Integrated with Google OAuth for secure and easy login.
